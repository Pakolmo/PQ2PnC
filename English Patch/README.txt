English version of Police Quest II: Point and Click. Only works in ScummVM.
REQUIRES Police Quest II version 1.002.011

- Copy PATCH into the PQ2 game folder.
- Add the game using the latest version of scummVM.
- Optional: Set MIDI sound to Roland in the game tab for better sound.
